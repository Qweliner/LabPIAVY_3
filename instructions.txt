Программа для сортировки данных об организациях

Соблюдайте следующие требования для корректной работы:

Путь к папке: Указывайте только существующие пути к папкам. Не используйте кириллицу в пути.

Имя файла: Вводите имя существующего файла output.txt, сгенерированного программой обработки корреспонденции и адресов организаций. Файл должен находиться в указанной папке.

Специальные символы: Не используйте специальные символы в имени файла.

Расширение файла: Расширение .txt указывать не требуется, программа предполагает, что входной файл имеет это расширение.

-----Порядок работы-----

Перед началом работы: Необходимо указать путь к папке, содержащей входной файл output.txt.

Главное меню:

"Сортировка данных": Переход к процессу сортировки данных об организациях. Доступно только после указания пути к папке.

"Путь к папке": Ввод пути к папке, где располагается входной файл output.txt.

"Инструкция": Отображение данного руководства пользователя.

"Выход": Завершение работы программы.

Процесс сортировки данных:

Входной файл: Программа запросит имя входного файла. По умолчанию ожидается файл output.txt, находящийся в указанной папке. Если файл называется иначе, введите его имя (без расширения .txt).

Выбор поля для сортировки: Выберите одно из доступных полей для сортировки:

Организация

Адрес

Контактное лицо

Выбор направления сортировки: Выберите направление сортировки:

По возрастанию

По убыванию

Esc: В любой момент можно вернуться в главное меню, нажав клавишу Esc.

-----Результат-----

Результат сортировки: Отсортированные данные будут сохранены в новом текстовом файле. Имя файла будет автоматически сгенерировано на основе выбранного поля сортировки и направления (например, sorted_by_Организация_asc.txt). Файл будет создан в указанной папке.

Информация в файле: В начале файла будет указано, по какому полю и в каком направлении была выполнена сортировка.
